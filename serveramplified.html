Existence SMP Community Server


Project Amplified is the latest Existence SMP Project Server, taking place on Minecraft 1.18 following the closure of the 1st Community Server in preperation for the 2nd.
The server features and amplified world type on the latest Caves and Cliffs Update Part II, along with a wide range of Minecraft 1.18 datapacks.
Visit our Discord and Twitter for the latest news and updates
