Project Keystone

Project Keystone was started back in May 2019 as an exclusive Patreon server, opening to trusted community members in January 2020.
This page contains rules and information about the project specificlly, for more general infromation about Project Keystone see the server page.

Existence SMP: Project Keystone is powered by PaperMC and hosted by CubedHost

Online Map
The server features an online dynamic map which can be viewed at map.existencesmp.com
